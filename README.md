# SimpleKitsPE

Usage:

/diamondkit - Get a Diamond Kit

/ironkit - Get a Iron Kit

/goldenkit - Get a golden Kit

/chainkit - Get a chain Kit

/leatherkit - Get a leather Kit

/foodkit - Get 10 apples Kit

/bowkit - Get a bow and 64 Arrows
